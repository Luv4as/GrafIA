# GrafIA

## Descrição
A IA funciona como mentor do aprendizado, onde questiona se o aluno tem as bases necessárias para entender a resposta daquele assunto. Interagindo e ensinando sobre as bases de conhecimento necessárias para enfim dar uma resposta para o aluno sobre a pergunta inicial, demonstrando o contexto das bases teóricas de estar respondendo daquela forma.
